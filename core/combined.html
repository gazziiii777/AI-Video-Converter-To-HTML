The Prusa Core One is an enclosed CoreXY 3D printer with a 250×220×270mm build volume. The Prusa Core One features an all-steel exoskeleton frame for stability and active chamber temperature control that reaches 55°C. The Prusa Core One prints at higher speeds than previous Prusa models while maintaining quality across materials including PLA, PETG, ASA, PC, and Nylon. This is achieved through factory-tuned print profiles for over 200 different filaments.

The Prusa Core One uses the Nextruder extruder system with 360° cooling that enables printing 75° overhangs without supports. This extruder system works alongside multiple sensors for automatic functionality. For example, the Prusa Core One includes a load cell for first-layer calibration, dual filament sensors for material detection, five thermistors for temperature monitoring, and a door sensor. Due to this comprehensive setup, the Prusa Core One requires only 10 minutes from unpacking to first print, guided by a step-by-step initialization process.

The Prusa Core One integrates with Prusa's complete ecosystem while maintaining offline functionality. This ecosystem includes PrusaSlicer software with one-click printing, Prusa Connect for remote management, and the Prusa App for mobile control. Through these connections, the Prusa Core One provides access to Printables.com with its library of 3D models. For long-term value, the Prusa Core One offers backward compatibility for accessories like print sheets and nozzles from previous models. Additionally, the Prusa Core One supports future expansions including Multi-Material Upgrade (MMU3) compatibility for printing with up to five filaments simultaneously.

The Prusa Core One prints with layer heights from 0.05mm to 0.30mm. The Prusa Core One achieves this precision through a CoreXY kinematic system with 0.9° stepper motors that reduce vertical fine artifacts (VFAs) on print surfaces. This reduction in VFAs results in smoother finished objects. The Prusa Core One incorporates the Nextruder's 360° cooling system, which enables printing of 75° overhangs without supports. This cooling capability maintains dimensional accuracy while reducing material waste during printing.

The Prusa Core One is Prusa's fastest printer due to three key components working together. For example, the Prusa Core One combines CoreXY kinematics, high-flow CHT (Compressed Heat Transmission) nozzle, and built-in Input Shaper technology. These components maintain print quality at higher speeds than previous Prusa models. To ensure consistent results, the Prusa Core One uses load cell-based automatic bed leveling that probes only the specific print area required. This targeted probing saves time while ensuring perfect first layers. Due to this probing system, users don't need to adjust Z-offset when switching between different build plates.

The mechanical design of the Prusa Core One features an all-steel exoskeleton frame for stability during rapid movements. This frame construction minimizes vibrations that could affect print quality. For precise height control, the Prusa Core One implements a Z-axis with three independent stepper motors. At the rear of the printer, two exhaust fans automatically adjust speed based on chamber temperature requirements. Additionally, the Prusa Core One includes phase stepping for smoother motor operation and fan motor monitoring to detect potential cooling issues before they affect prints.

In controlled testing, the Prusa Core One produces consistent glossy surface finishes throughout prints, particularly with technical materials. The enclosed chamber of the Prusa Core One reaches temperatures up to 55°C and maintains this thermal stability during printing. When printing a test benchy model, the Prusa Core One completes the print in 37 minutes while maintaining accurate time estimates on the display. This accuracy in time estimation makes print scheduling predictable. For high-temperature materials, the temperature regulation system automatically preheats the chamber before starting prints, ensuring proper layer adhesion from the first layer onward.

Which filament materials can you use with the Prusa Core One?

The Prusa Core One supports these materials without additional accessories: PLA, PETG, Flex, PVA, PC, PP, CPE, and PVB. The Prusa Core One can also handle ABS, ASA, HIPS, and PA (Nylon) when equipped with the optional Advanced Filtration System. To accommodate these materials, the Prusa Core One's hot end reaches temperatures up to 290°C, and the heated bed operates up to 120°C. This temperature range enables printing across all these material types effectively.

The Prusa Core One features an enclosed chamber that creates optimal conditions for technical materials. For high-temperature filaments, the Prusa Core One automatically preheats the chamber to 40°C before starting prints. During operation, this chamber temperature reaches up to 55°C, which prevents warping and layer separation in materials like ABS and ASA. For lower-temperature materials such as PLA, the Prusa Core One maintains chamber temperatures between 20-30°C. This temperature control is achieved using two exhaust fans that adjust speed automatically. Additionally, the top panel of the Prusa Core One includes a manual vent that opens for PLA printing and closes for high-temperature materials.

The Prusa Core One works with any standard 1.75mm filament from third-party manufacturers. Prusa also offers their own Prusament line with 200+ pre-configured material profiles in PrusaSlicer for the Prusa Core One. To enhance material compatibility, the Prusa Core One's hardware includes specific features. For example, the Prusa Core One uses a planetary gearbox with 10:1 ratio for flexible filaments and dual filament sensors (one at entry and one at the hot end). The Prusa Core One comes with a high-flow Compressed Heat Transmission (CHT) brass nozzle for standard materials. This stock brass nozzle handles non-abrasive materials, while optional hardened steel nozzles are available for carbon fiber-reinforced filaments. For convenience, the nozzle change system allows users to switch between nozzle types without requiring hot tightening.

What is the build volume of the Prusa Core One?

The Prusa Core One has a build volume of 250 × 220 × 270 mm (9.8 × 8.6 × 10.6 inches). The Prusa Core One's build volume represents a 30% increase in total print volume compared to the previous Prusa i3 Mark 4S (MK4S) printer. This increase comes from the Prusa Core One adding 10 mm to the Y-axis and 50 mm to the Z-axis while maintaining the same 250 mm width on the X-axis as the MK4S.

The Prusa Core One has external dimensions of 415 × 444 × 555 mm (16.3 × 17.5 × 21.8 inches). The Prusa Core One is 50% smaller than an MK4S with an enclosure despite providing more print volume. This space efficiency is achieved through the compact CoreXY design that eliminates the moving bed found in i3-style printers. Additionally, the sides of the Prusa Core One feature recessed pockets. These recessed pockets reduce interior volume while providing space for the spool holder and storage.

What printer controls are available on the Prusa Core One?

The Prusa Core One features a 3.5-inch graphic 65k color touchscreen with a rotary dial interface for navigation. The Prusa Core One's screen mounts on a magnetic panel that detaches if the printer is tilted, preventing display damage. For status indication, the Prusa Core One includes an LED strip on the front edge of the build plate. This LED strip displays different colors to show printer status: idle, printing, warning, or error. Through the touchscreen interface, users access direct operation functions. These functions include filament loading/unloading, print job controls (start/pause/stop), and manual Z-offset adjustments during first layer printing.

The Prusa Core One provides multiple calibration tools through the on-device interface. During initial setup, the Prusa Core One runs a self-test sequence that verifies component functionality. For consistent print quality, the Prusa Core One implements an automatic bed leveling system. This bed leveling system uses a load cell sensor to measure the build surface before each print, which eliminates the need for manual Z-offset adjustments when switching print sheets. From the display, users can control additional settings. For example, users can adjust chamber temperature settings, disable the door sensor that pauses prints when opened, and access maintenance functions. To ensure reliable printing, the Prusa Core One automatically detects filament presence using two sensors: one sensor at the filament entry point and another sensor at the extruder.

What connectivity options are available on the Prusa Core One?

The Prusa Core One provides three primary connection methods: Ethernet port for wired network access, detachable ESP Wi-Fi module, and USB port for direct drive access. The Prusa Core One's Wi-Fi module can be physically disconnected for environments requiring heightened security. This disconnection still maintains full printer functionality through the other connection methods. For initial setup, the Prusa Core One includes an NFC (Near Field Communication) reader on the back panel. This NFC reader transfers network credentials from a smartphone to the printer with a simple tap.

The Prusa Core One operates with both local and cloud network protocols. Prusa Link allows connection through the local network without internet dependency, while Prusa Connect provides cloud-based remote access with additional features. Both these services work with PrusaSlicer software for direct print file transmission. The Prusa Core One is compatible with computers through web browsers and with mobile devices. This compatibility extends to the dedicated Prusa App for iOS and Android. For monitoring capabilities, an optional USB-C camera ($40) mounts in the upper left corner of the print chamber. The USB-C camera connects directly to the Prusa Core One's motherboard but requires an active Wi-Fi connection and Prusa Connect service to function. This connection requirement exists because the camera does not stream video through the local network.

What is the design of the Prusa Core and build quality of the Prusa Core One?

The Prusa Core One uses an all-steel exoskeleton frame with a matte finish sheet metal enclosure. The Prusa Core One weighs 22.5 kg and measures 415×444×555 mm (16.3×17.5×21.8 inches). For visibility, the Prusa Core One includes polycarbonate panels that attach with plastic rivets. The frame of the Prusa Core One features orange-colored accents on the door handle, knob, and select internal components. For maintenance and modification, the enclosure panels are not glued, allowing complete disassembly with standard screwdrivers.

The Prusa Core One incorporates functional design elements for practical use. For space efficiency, the Prusa Core One includes two recessed side pockets that reduce interior volume while providing utility. One of these pockets houses the filament spool holder while the other offers storage space. For flexibility in positioning, the front door opens 180° and can be configured to hinge from either left or right side. To prevent damage, a magnetic mount secures the display panel, detaching if the printer is tilted. For temperature management, the top panel features a manual sliding vent when printing different materials. As a visual indicator, an LED strip on the front edge of the bed displays printer status through color coding: idle, printing, warning, or error.

The Prusa Core One uses a CoreXY motion system with 0.9° stepper motors on X and Y axes. This motor choice reduces vertical fine artifacts in printed objects. For precision movement, the X-axis rides on a linear rail mounted to a steel extrusion, while the Y-axis uses 10mm smooth rods. For Z-axis movement, the Prusa Core One employs three independent stepper motors with 8mm smooth rods. This CoreXY design eliminates the moving bed found in i3-style printers, which increases stability at higher speeds. For heat resistance and durability, internal structural components use 3D-printed carbon fiber polycarbonate blend parts rather than standard PETG. To secure the extruder system, the print head mounts to the X-axis rail via an aluminum adapter plate that holds the Nextruder with its planetary gearbox.
